01 · Geometria refinada

Função paramétrica isósceles · um único slope para os 3 triângulos

A vetorização da exploração anterior teve pequenos desalinhamentos por slopes diferentes entre M1, M2 e M3. Reconstruí os 3 triângulos a partir de uma única razão half/h = 26/48 = 0.5417. Os ápices ficam exatamente no ponto médio das bases (snap em inteiros), com baseline única em y=64. A bandeirinha está centralizada em x=50.

// função paramétrica isósceles
function tri(cx, h, baseY=64) {
  const half = Math.round(h * 0.5417);
  const x1 = cx - half;
  const x2 = cx + half;
  const apexY = baseY - h;
  return `M${x1} ${baseY}
          L${cx} ${apexY}
          L${x2} ${baseY}Z`;
}

// os 3 oficiais
tri(50, 48)  // M1 · "A" do Auge
tri(74, 26)  // M2 · direita
tri(28, 22)  // M3 · fundo
